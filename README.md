# CodeAlpha_Simple-E-Commerce-Store
Simple E-Commerce Store 🛒

A full-stack Simple E-Commerce Store web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. This project was created as part of the CodeAlpha Internship Program to demonstrate the implementation of a modern online shopping platform with frontend, backend, and database integration.

The application allows users to browse products, register/login securely, add products to the shopping cart, and place orders through a responsive and user-friendly interface. The project follows a clean folder structure and includes REST API integration between the frontend and backend.

🚀 Features

User Registration and Login Authentication

Product Listing with Images and Prices

Product Details Page

Shopping Cart Functionality

Add to Cart and Remove from Cart

Order Processing System

MongoDB Database Integration

Responsive Modern UI Design

Backend API using Express.js

Automatic Dummy Product Seeding

Mobile-Friendly Layout

⚙️ Installation and Setup

Clone the Repository

git clone <your-github-repository-link>

Install Backend Dependencies

cd backend
npm install

Start MongoDB Community Server

Make sure MongoDB is running locally on:

mongodb://localhost:27017/ecommerce

Run the Backend Server

npm start

Open the Application

http://localhost:5001
